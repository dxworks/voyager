# Dxworks Voyager

This project was generated using the `dxworks-template-node-ts` repository template.
## Installation

Use `npm` to install 

```bash
npm i -g @dxworks/voyager
```

## Usage

```shell
dx-voyager
```
or
```shell
dxw voyager
```
