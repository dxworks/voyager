# Dxworks Voyager

This project is a work in progress. This represents the second version of Voyager.

## Installation

Use `npm` to install

```bash
npm i -g @dxworks/voyager
```

## Usage

```shell
dx-voyager
```
or
```shell
dxw voyager
```
